1. String Length

Write a Python program that takes a string as input and prints the length of the string.

        
# Input string
input_string = input("Enter a string: ")

# Calculate length
length = len(input_string)

# Display result
print(f"The length of the string is: {length}")

2. Vowel Count

Write a Python program that counts the number of vowels in a given string.

        
# Input string
input_string = input("Enter a string: ").lower()

# Define vowels
vowels = "aeiou"

# Count vowels
vowel_count = sum(1 for char in input_string if char in vowels)

# Display result
print(f"The number of vowels in the string is: {vowel_count}")

3. Palindrome Check

Write a Python program that checks if a given string is a palindrome (reads the same forwards and backwards).

        
# Input string
input_string = input("Enter a string: ")

# Check if palindrome
is_palindrome = input_string == input_string[::-1]

# Display result
if is_palindrome:
    print(f"{input_string} is a palindrome.")
else:
    print(f"{input_string} is not a palindrome.")

5. Character Frequency

Write a Python program that counts the frequency of each character in a string.

        
# Input string
input_string = input("Enter a string: ")

# Count character frequency
char_frequency = {}
for char in input_string:
    if char in char_frequency:
        char_frequency[char] += 1
    else:
        char_frequency[char] = 1

# Display result
print("Character frequency:")
for char, frequency in char_frequency.items():
    print(f"{char}: {frequency}")

6. Remove Duplicates

Write a Python program that removes duplicate characters from a string.

        
# Input string
input_string = input("Enter a string: ")

# Remove duplicates
unique_chars = ""
for char in input_string:
    if char not in unique_chars:
        unique_chars += char

# Display result
print(f"String without duplicates: {unique_chars}")

7. Replace Substring

Write a Python program that replaces a substring within a string with another substring.

        
# Input string and substrings
input_string = input("Enter a string: ")
old_substring = input("Enter the substring to replace: ")
new_substring = input("Enter the new substring: ")

# Replace substring
modified_string = input_string.replace(old_substring, new_substring)

# Display result
print(f"Modified string: {modified_string}")
